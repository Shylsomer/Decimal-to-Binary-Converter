# Decimal to Binary Conversion Program
This program retrieves number inputs from the user, converts them into binary, and outputs the result. Does NOT take/convert IPv4's, ASCII characters, decimal fractions, or negative numbers. It will not work correctly if those are used as input.

---

## How to Run
### In a terminal

Run the program via the command below.
```
   python main.py
```
### In Google Colab

1. Under "Insert" click "Code Cell".
2. Paste the code in main.py into your code cell.
3. Click the play button.
